This is the workplace for designing a buck converter with 6.8V and 20A current max.
This circuit is used for serving the 12 servo Miuzel 20kg torque with a current load from 1-2A 
The purpose of using 12 servos is to design a 12 servo with enough power supply and connect it to the Jetson Orin Nano to control the movement
The input power source is only designed for a 5s LiPo battery.
